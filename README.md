# ShootingGameProject
팀 스파르타 내일배움캠프 유니티 게임 개발 입문 주차 프로젝트
## 🖥️ 프로젝트 소개
우주 배경을 기반으로한 2D 스크롤 방식 비행 슈팅 게임   
적 비행기의 총알을 피하고 적을 파괴하여 점수를 올리는 게임

## 👨‍👨‍👦 멤버구성
- 김보근 (팀장) : UI, 보스 
- 최세은 : UI, 총알, 애니메이션
- 김창민 : 스킬
- 박창우 : UI, 사운드

## 🕶 기본 조작법
AWSD : 이동   
C : 실드 스킬 (앞으로 날아가면서 총알, 적을 막아줌)   
F : 전체 스킬 (모든 적이 사라짐)
